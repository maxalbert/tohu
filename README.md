tohu: Creating complex random data with ease
============================================

Prerequisites
-------------

- [pipenv](https://docs.pipenv.org/)
- [PostgreSQL](https://www.postgresql.org/) (for running the tests)


Quick start
-----------

To run the tests:
```
$ pipenv install --dev
$ pipenv shell
$ make test
```
